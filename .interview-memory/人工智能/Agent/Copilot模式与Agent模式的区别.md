# Copilot 模式与 Agent 模式的区别

## 速记卡片
`协 | 人协自，辅对代 | 模式对比`

## 原始知识
- Copilot（副驾驶）模式：
  - **人**（人类主导）- 人做决策，AI辅助
  - **协**（协同）- 人机协作，AI提供建议
  - **辅**（辅助）- 辅助人类完成任务
- Agent（代理）模式：
  - **自**（自主）- AI自主决策执行
  - **代**（代理）- AI代理完成任务
  - 人只需设定目标，不参与中间过程
- **易混点**：两者不是非此即彼，可以结合
- **追问点**：什么时候用Copilot，什么时候用Agent？

## 顺口溜
**人协自，辅对代**

## 单字锚点 + 防崩推导
- **锚点字**：**协**
- **扩展口诀**：**人协自，辅对代**
- **为什么选它**：Copilot的核心是"协同"，"协"是记忆钩子。
- **防崩推导**：
  - 协（协同）→ Copilot是人机协同
  - 人（人类）→ 人类主导
  - 自（自主）→ Agent是自主执行
  - 辅（辅助）→ Copilot辅助人类
  - 代（代理）→ Agent代理完成
  - 对 → 两者对比

## 一字一画面
- **人** —— 像**司机开车**：AI坐在副驾指路，但方向盘在人手里。（本质：人类主导）
- **协** —— 像**双人划艇**：两人配合，但船长决定方向。（本质：人机协同）
- **自** —— 像**自动驾驶**：设定目的地，车自己开。（本质：自主执行）
- **代** —— 像**代驾**：你喝酒了，代驾帮你开回家。（本质：代理完成）

## 面试话术
- 问区别 → "Copilot和Agent的区别口诀是**人协自，辅对代**。**人**——Copilot是人类主导；**协**——人机协同，AI提供建议；**辅**——AI辅助人类完成任务；**自**——Agent是AI自主决策；**代**——AI代理完成全部任务。Copilot适合需要人工审核的场景（如代码审查），Agent适合自动化任务（如定时报表生成）。实际产品中两者常结合：复杂任务用Agent，关键环节用Copilot审核。"

## 防混补丁
- 容易觉得Agent更好。其实各有适用场景。补丁：**Copilot是副驾驶，Agent是代驾**。"
