# 什么是 Agent？与大模型有什么本质不同？

## 场景触发词
Agent 是什么、Agent vs LLM、智能体本质、大模型和 Agent 的区别

## 速记卡片
`代 | 代决工，三不同 | Agent vs LLM`

## 原始知识
- **大模型**：会说话的“大脑”，核心能力是理解、生成、推理
- **Agent**：有大脑的“行动体”，核心能力是感知→决策→执行→反馈
- **本质不同点**：
  1. **主动性**：LLM 被动响应；Agent 主动规划、调用工具
  2. **工具性**：LLM 只有文本输入输出；Agent 可操作外部工具/API
  3. **记忆性**：LLM 单次对话无状态；Agent 有长期/短期记忆、可持久化
  4. **循环性**：LLM 一问一答；Agent 是“观察-思考-行动”的闭环

## 顺口溜
**代决工，三不同**

## 单字锚点 + 防崩推导
- **锚点字**：代
- **扩展口诀**：代决工
- **为什么选它**：“代”= Agent 是代替人类执行任务的实体，最能体现本质
- **防崩推导**：
  1. 代 → 代替人干活
  2. 决 → 不是简单回复，而是自主决策
  3. 工 → 会调用工具完成目标
  4. 三不同 → 主动性、工具性、循环性（比大模型多了执行层）

## 一字一画面
**代** —— 像 **一个机器人管家**：[深蓝色] 听到主人说“明天下雨”，不是只回一句“带伞”，而是主动查天气、约车、设闹钟、发提醒，完成一连串动作。
**决** —— 像 **十字路口的红绿灯**：[黄灯闪烁] 面对多个选择，Agent 自己判断走哪条路，而不是等人类指挥。
**工** —— 像 **瑞士军刀**：[银灰色展开] Agent 随身携带各种工具（搜索、代码执行、发邮件），按需使用。

本质：Agent = 大模型（大脑）+ 工具（手脚）+ 记忆（经验）+ 循环控制（工作流）

## 面试话术
Agent 是在大模型基础上的“行动封装体”。大模型像大脑，只能给你回答；Agent 像长了手脚的大脑，能自己决定调用什么工具、分几步完成任务，并把结果反馈回来继续优化。核心区别就三点：**主动决策、工具调用、循环执行**。

## 防混补丁
**别混**：LLM 是 Agent 的核心组件，但 Agent 不等于 LLM。口诀：**大脑不是人，有手才成事**。
