# Agent 推理模式有哪些？

## 场景触发词
Agent 推理模式、大模型推理方式、Agent 怎么思考、LLM 推理策略

## 速记卡片
`模 | 模链计树反，五模式 | Agent 推理模式`

## 原始知识
Agent 常见的五种推理模式（由简到繁）：

1. **直接推理（Direct Reasoning）**
   - 直接给答案，不展开中间过程
   - 适合简单问答、事实查询

2. **链式推理 / CoT（Chain of Thought）**
   - 把思考过程拆成一步一步的链
   - 让模型"自言自语"推导出答案
   - 适合数学、逻辑、多步推理任务

3. **计划-执行（Plan-and-Execute）**
   - 先制定完整计划，再按步骤执行
   - 计划阶段一次性想清楚，执行阶段专注行动
   - 适合多步骤复杂任务

4. **思维树 / ToT（Tree of Thoughts）**
   - 维护多个推理分支，像树一样探索
   - 评估各路径，选择最优解
   - 适合需要探索、决策、创造性任务

5. **反思-行动 / ReAct（Reasoning + Acting）**
   - 推理与外部行动交替循环
   - 观察环境 → 思考 → 行动 → 再观察
   - 最完整的 Agent 推理模式，能与工具结合

- **锚点字**：模

  2. 链 → **链**式推理（CoT），一步一步想
  3. 计 → **计**划-执行（Plan-and-Execute），先想后做
  4. 树 → 思维**树**（ToT），多路探索
  5. 反 → **反**思-行动（ReAct），边想边做，循环迭代

## 一字一画面
**模** —— 像 **五种武术招式**：[五本秘籍] 从简到繁：直拳（直接推理）、连环腿（CoT）、战术板（计划-执行）、迷宫探路（思维树）、实战格斗（ReAct）。
**链** —— 像 **多米诺骨牌**：[白色骨牌] 一块推倒一块，必须按顺序才能到达终点。
**计** —— 像 **藏宝图**：[羊皮纸] 先画好路线图，再按图索骥挖宝。
**树** —— 像 **分叉的地下通道**：[多条岔路] 每走一段就评估哪条路更有希望，逐步排除死路。
**反** —— 像 **格斗中的见招拆招**：[拳脚交错] 打出一招，观察对手反应，再想下一招，循环往复。

## 面试话术
Agent 的推理模式主要有五种，由简到繁：直接推理（最简单，直接给答案）、链式推理 CoT（一步一步想）、计划-执行（先定计划再做）、思维树 ToT（多路探索择优）、以及反思-行动 ReAct（边想边做、与工具结合）。实际项目中，简单任务用 CoT，复杂任务用 Plan-and-Execute 或 ReAct，探索性问题用 ToT。

## 防混补丁
**别混**：不要把 CoT 和 ReAct 搞混。CoT 是纯文本推理链，ReAct 是推理+行动+观察的循环。口诀：**CoT 纸上谈兵，ReAct 真刀真枪**。
