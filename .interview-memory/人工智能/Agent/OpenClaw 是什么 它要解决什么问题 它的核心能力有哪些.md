# 单字锚点 + 防崩推导

## 锚点字：爪

- **扩展口诀**：爪（OpenClaw）长（长对话记忆）工（工作流/工具调用）三核心
- **为什么选它**："爪"是OpenClaw的品牌字，一眼识别，绝不混淆
- **推导链**：
  1. 看到"爪" → 想到 OpenClaw
  2. OpenClaw 是**开源Agent框架** → 解决什么问题？Agent开发太复杂
  3. "长" → **长对话记忆**问题、**Context Window**管理
  4. "工" → **工具调用** + 复杂**工作流**
  5. "三核心" → 实际上有五大能力，但"三"是顺口：上下文管理、记忆管理、工具调用是三大基础，Runner和Compaction是两大进阶保障

# 一字一画面

- **爪** —— 像[机械爪]：银灰色三指机械爪精准夹起散落的零件，组装成一台完整机器人（本质：OpenClaw是组装Agent的框架）
- **长** —— 像[卷尺拉出三米]：黄色卷尺"哗啦"拉长到极限，刻度数字不停跳动，最后自动折叠回收（本质：长对话记忆需要自动管理）
- **工** —— 像[工人腰间工具带]：蓝色工装裤上挂满扳手螺丝刀，工人随手一抽就能用（本质：工具调用即插即用）
