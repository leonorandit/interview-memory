# LangChain 的核心组件

## 速记卡片
`件 | 模提链，代记器 | 六大组件`

## 原始知识
- LangChain六大核心组件：
  1. **模**（Model）- 封装LLM和Chat Model
  2. **提**（Prompt）- Prompt模板和管理
  3. **链**（Chain）- 组件串联流水线
  4. **代**（Agent）- 自主决策、调用工具的代理
  5. **记**（Memory）- 对话历史管理
  6. **器**（Document Loader / Vector Store / Tool）- 文档加载、向量存储、外部工具
- **易混点**：Chain vs Agent
- **追问点**：各组件怎么配合？Memory有哪些类型？

## 顺口溜
**模提链，代记器**

## 单字锚点 + 防崩推导
- **锚点字**：**件**
- **扩展口诀**：**六组件，模提链代记器**
- **为什么选它**：面试题问"核心组件"，"件"直接对应。
- **防崩推导**：
  - 件（组件）→ LangChain的六大组件
  - 模（Model）→ LLM/Chat Model封装
  - 提（Prompt）→ Prompt模板
  - 链（Chain）→ 串联流水线
  - 代（Agent）→ 自主代理
  - 记（Memory）→ 对话记忆
  - 器（工具/存储）→ 文档加载、向量库、外部工具

## 一字一画面
- **模** —— 像**发动机**：提供动力。（本质：LLM模型）
- **提** —— 像**指令单**：告诉发动机怎么转。（本质：Prompt模板）
- **链** —— 像**传送带**：把零件从一个工位传到下一个。（本质：Chain串联）
- **代** —— 像**智能调度员**：自己决定下一步做什么。（本质：Agent自主决策）
- **记** —— 像**记事本**：记住之前说了什么。（本质：Memory对话历史）
- **器** —— 像**工具箱和仓库**：放工具和存货。（本质：工具+向量存储）

## 面试话术
- 问核心组件 → "LangChain有六大核心组件，口诀是**模提链，代记器**。**模**——Model，封装各种LLM；**提**——Prompt，模板化管理提示词；**链**——Chain，将组件串联成流水线；**代**——Agent，自主决策调用工具；**记**——Memory，管理对话历史；**器**——Document Loader、Vector Store、Tool等，负责文档加载、向量存储和外部工具集成。"

## 防混补丁
- 容易分不清Chain和Agent。Chain是固定流程，Agent是动态决策。补丁：**Chain是流水线，Agent是调度员**。"
