# 大模型 alignment（对齐）是什么？

## 速记卡片
`对 | 对三H，人奖直，有税交 | 对齐概念与路线`

## 原始知识
- **对齐（Alignment）**：让模型行为与人类价值观、意图、偏好保持一致
- **为什么需要对齐**：预训练只学语言模式，不懂"好/坏"；可能输出有害、偏见、欺骗性内容
- **3H 目标**：Helpful（有用）、Harmless（无害）、Honest（诚实）
- **RLHF**：人类反馈强化学习，SFT → 训练 Reward Model → PPO 优化
- **DPO**：直接偏好优化，跳过显式 Reward Model，直接用偏好对优化策略
- **对齐税（Alignment Tax）**：对齐后模型在某些能力上下降（知识性
