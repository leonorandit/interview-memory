# 什么是护栏技术

## 速记卡片
`栏 | 入出检，安全审 | Guardrails`

## 原始知识
- 护栏技术（Guardrails）：在LLM输入输出两端设置安全控制，防止有害/不合规内容
- 核心功能：
  1. **入**（输入检查）- 过滤恶意Prompt、注入攻击
  2. **出**（输出检查）- 过滤有害内容、敏感信息泄露
  3. **检**（检测）- 毒性检测、偏见检测、幻觉检测
  4. **安全**（安全防护）- 防止越狱、提示注入
  5. **审**（合规审查）- 符合法规、企业政策
- 工具：NeMo Guardrails、Guardrails AI、Llama Guard
- **易混点**：护栏 vs 内容审核
- **追问点**：怎么防止提示注入？怎么检测幻觉？

## 顺口溜
**入出检，安全审**

## 单字锚点 + 防崩推导
- **锚点字**：**栏**
- **扩展口诀**：**入出检，安全审**
- **为什么选它**：Guardrails直译就是"护栏"，"栏"是天然记忆钩子。
- **防崩推导**：
  - 栏（护栏）→ 在LLM周围设置安全护栏
  - 入（输入检查）→ 检查用户输入是否恶意
  - 出（输出检查）→ 检查模型输出是否合规
  - 检（检测）→ 毒性、偏见、幻觉检测
  - 安全 → 防止越狱和注入
  - 审（审查）→ 合规审查

## 一字一画面
- **栏** —— 像**高速公路护栏**：防止车冲出路面。（本质：安全防护）
- **入** —— 像**安检门**：进门先检查有没有带违禁品。（本质：输入过滤）
- **出** —— 像**出口质检**：产品出厂前检查是否合格。（本质：输出过滤）
- **审** —— 像**法务审核**：合同发布前法务先看一遍。（本质：合规审查）

## 面试话术
- 问什么是护栏技术 → "护栏技术是在LLM输入输出两端设置安全控制，口诀是**入出检，安全审**。**入**——输入检查，过滤恶意Prompt和注入攻击；**出**——输出检查，防止有害内容和敏感信息泄露；**检**——毒性、偏见、幻觉检测；**安全**——防止越狱攻击；**审**——合规审查。常用工具有NeMo Guardrails、Guardrails AI、Llama Guard。"

## 防混补丁
- 容易和"内容审核"搞混。内容审核是人工或规则过滤，护栏是系统化的多层安全控制。补丁：**审核是保安，护栏是安防系统**。
