# Transformer 的位置编码是怎样的

## 速记卡片
`位 | 位弦旋，学R加 | 绝对·相对·旋转`

## 原始知识
- 原始Transformer：正弦余弦绝对位置编码
- 可学习位置编码：BERT/GPT-1直接学习位置向量
- RoPE旋转位置编码：LLaMA等主流模型使用，旋转矩阵注入相对位置信息
- ALiBi：注意力偏置，外推性好
- 相对位置编码：T5等使用
- **易混点**：绝对编码和相对编码的区别

## 顺口溜
位弦旋，学R加

## 单字锚点 + 防崩推导
- **锚点字**：**位**
- **扩展口诀**：**位置弦**
- **为什么选它**："位"直接对应"位置编码"，是概念的核心字。
- **防崩推导**：
  - 位 → 位置编码 → 给token标座位号
  - 弦 → 正弦余弦 → 原始方案，固定波形
  - 旋 → 旋转变换 → RoPE，现在最流行
  - 学 → 可学习 → BERT/GPT学出来的位置向量
  - R → RoPE → 旋转位置编码
  - 加 → 加到输入 → 位置向量+词向量

## 一字一画面
- **位** —— 像**座位号**：每个token都要知道自己坐第几排。（位置信息）
- **弦** —— 像**波浪楼梯**：正弦余弦画出上下起伏的台阶，每个台阶高度不同。（唯一标识）
- **旋** —— 像**旋转木马**：每个token坐在木马上，位置越远转过的角度越大。（RoPE）
- **学** —— 像**买票选座**：模型自己学习每个座位值多少钱。（可学习编码）
- **R** —— 像**罗盘**：旋转角度决定方向，角度差就是位置差。（相对性）
- **加** —— 像**加调料**：位置向量加到词向量上，一起喂进模型。（输入融合）

## 面试话术
- "位置编码经历了三代演进，口诀是**位弦旋，学R加**。原始Transformer用正弦余弦绝对编码；BERT用可学习位置编码；现在主流是RoPE旋转位置编码，通过旋转矩阵把相对位置信息注入Attention，外推性更好。都是加到输入Embedding上使用的。"

## 防混补丁
- 绝对和相对容易混。补丁：**绝对编码说'我在第5排'，相对编码说'我离他3排远'，RoPE两者兼得**。

## 追问预演
- **Q：为什么Transformer需要位置编码？** A：Self-Attention本身是位置无关的，打乱序列顺序不影响Attention计算，必须显式注入位置信息。
- **Q：RoPE为什么外推性好？** A：RoPE通过旋转矩阵编码相对位置，旋转角度和位置差成正比，长度外推时只需要旋转更大的角度，不需要重新训练位置编码。
