# Word2Vec 到 BERT 有怎样的改进？

## 速记卡片
`深 | 深双向，掩码训 | 浅静态→深动态`

## 原始知识
- **Word2Vec**：浅层网络，静态词向量，一词一义
- **BERT**：深层Transformer，动态上下文表示，一词多义
- **改进1-深度**：Word2Vec无隐藏层或很浅，BERT用12/24层Transformer
- **改进2-双向**：Word2Vec是伪双向（拼接），BERT是真双向（自注意力）
- **改进3-预训练任务**：MLM掩码语言模型 + NSP下一句预测
- **改进4-上下文**：Word2Vec局部窗口，BERT全局 attention
- **改进5-子词**：BERT用WordPiece，解决OOV问题
- **易混点**：BERT不是ELMo的简单加深，架构从LSTM换到Transformer

## 顺口溜
**深双向，掩码训**

## 单字锚点 + 防崩推导
- **锚点字**：**深**
- **扩展口诀**：**深双掩，动全子**
- **为什么选它**："深"是BERT相比Word2Vec最直观的差异——网络深、表示深、理解深
- **防崩推导**：
  - 深 → 深层Transformer → 12/24层 vs Word2Vec的浅层
  - 双 → 真双向 → 自注意力同时看左右
  - 掩 → MLM掩码训练 → 完形填空式预训练
  - 动 → 动态向量 → 同一词不同句子不同表示
  - 全 → 全局attention → 不受窗口限制
  - 子 → WordPiece子词 → 解决未登录词

## 一字一画面
- **深** —— 像**浅池塘 vs 深海底**：Word2Vec只在表面游，BERT能潜到深层语义。（本质：网络深度）
- **双** —— 像**全景摄像头**：同时看清左右两边，不是左眼右眼分开拍。（本质：真双向）
- **掩** —— 像**完形填空考试**：遮住一个词让模型猜，强迫它理解上下文。（本质：MLM训练）
- **子** —— 像**乐高积木拼单词**：unhappiness拆成un+happi+ness，没见过整词也能拼。（本质：子词拆分）

## 面试话术
- "从Word2Vec到BERT，可以说是**深双向，掩码训**的飞跃。Word2Vec是浅层静态向量，一个词只有一个embedding；BERT是深层Transformer，用MLM掩码训练，真正双向，同一个词在不同上下文里表示完全不同。"
- "另外BERT用WordPiece处理子词，解决了OOV问题，而且全局attention不受窗口限制，语义捕捉能力远超Word2Vec的局部窗口。"

## 防混补丁
- 容易认为"BERT就是深度版Word2Vec"。补丁：**BERT和Word2Vec不是简单深浅关系，是架构革命——从局部预测到全局注意力，从静态查表到动态编码**。

## 追问预演
- **问**：BERT的MLM和GPT的自回归有什么区别？
  - **答**：MLM是遮住中间词看两边猜，适合理解任务；GPT是只看左边生成右边，适合生成任务。
- **问**：为什么BERT要用NSP？后来为什么很多模型又弃用了？
  - **答**：NSP初衷是学句子关系，但后来发现它对下游任务提升有限，且正样本构造太简单，RoBERTa等直接去掉NSP效果更好。
