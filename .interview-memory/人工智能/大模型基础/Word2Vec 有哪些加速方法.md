# Word2Vec 有哪些加速方法

## 速记卡片
`速 | 负采样分层加剪枝，哈夫曼树降复杂度 | 训练优化三板斧`

## 原始知识
- **负采样（Negative Sampling）**：只更新正例和K个负例，O(K)替代O(V)
- **层次Softmax（Hierarchical Softmax）**：霍夫曼树二分类路径，O(logV)
- **高频词下采样（Subsampling）**：抛弃高频词如"the""a"的部分出现，减少噪声
- **霍夫曼树优化**：高频词短路径，低频词长路径，最小化平均计算
- **并行化**：COWORD2VEC多线程，不同线程处理不同句子
- **加粗易混点**：下采样是扔高频词，不是扔低频词；负采样是构造负例，不是扔负例

## 顺口溜
负采分层下采样，三管齐下训练快

## 单字锚点 + 防崩推导
- **锚点字**：**速**
- **扩展口诀**：**速靠负层下**
- **为什么选它**：题目问的就是"加速方法"，"速"直接回应问题，同时"负层下"三个字对应三大方法。
- **防崩推导**：
  - 速 → 负 → 负采样 → 每轮只更K个负例+1个正例
  - 速 → 层 → 层次softmax → 霍夫曼树二分类 → O(logV)
  - 速 → 下 → 高频词下采样 → 扔掉"the"的80%出现 → 减少无意义更新
  - 速 → 外加并行 → 多线程分句子训练

## 一字一画面
- **速** —— 像**高速收费站开ETC**：原来每个车都要人工缴费（全词表softmax），现在自动扣费快速通行（负采样/层次softmax）。（本质：计算瓶颈突破）
- **负** —— 像**只抓几个嫌疑犯示众**：不用把全城人审一遍，抓几个典型就够了。（本质：采样近似）
- **下** —— 像**清理重复的垃圾邮件**：邮箱里"促销"出现一千次，只保留几次，其他直接删除。（本质：高频词降噪）

## 面试话术
- "Word2Vec的加速方法主要有三个。第一是**负采样**，把多分类softmax改成二分类，每轮只更新正例和几个负例；第二是**层次softmax**，用霍夫曼树把复杂度从O(V)降到O(logV)；第三是**高频词下采样**，对'the'这种高频词随机丢弃部分出现，减少无意义的参数更新。这三个方法可以同时用，训练速度能提升几十倍。"

## 防混补丁
- 容易把"下采样"理解成降低采样率。补丁：**下采样=Subsampling frequent words，专指丢弃高频词，不是欠采样**。

## 追问预演
- Q：负采样和层次softmax能一起用吗？
  - A：不能同时用于同一个输出，二者是互替方案；但都可以和下采样共用。
- Q：高频词下采样的阈值怎么定？
  - A：按词频概率丢弃，公式P(w) = 1 - sqrt(t/f(w))，t通常10^-5。
- Q：Skip-gram和CBOW哪个更需要加速？
  - A：Skip-gram，因为它的上下文窗口产生更多训练样本。
