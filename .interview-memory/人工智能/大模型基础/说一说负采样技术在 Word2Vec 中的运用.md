# 说一说负采样技术在 Word2Vec 中的运用

## 速记卡片
`负 | 负采样，噪声对比，少数正例多数造 | 二分类替多分类`

## 原始知识
- **问题背景**：softmax分母要遍历全词表V，训练极慢
- **核心思想**：把多分类变成二分类——区分"正样本对"和"负样本对"
- **正样本**：中心词和真实上下文词 → label=1
- **负样本**：中心词和随机抽的词（按词频^0.75分布）→ label=0
- **损失函数**：对正样本最大化log(sigmoid)，对负样本最小化log(sigmoid)
- **加粗易混点**：负采样只在训练时用，推理时不存在"负样本"概念

## 顺口溜
噪声对比，正负二分类

## 单字锚点 + 防崩推导
- **锚点字**：**负**
- **扩展口诀**：**负采样噪声对**
- **为什么选它**：这项技术的名字就叫"负采样"，"负"是题眼，正负对比是机制核心。
- **防崩推导**：
  - 负 → 负采样 → 不更新全词表，只抽K个负例
  - 负 → 正例是真实搭配 → 负例是随机拼凑
  - 负 → 损失 = logσ(v'·v) + Σlogσ(-v_neg·v) → 拉近正例、推远负例
  - 负 → 采样分布 ∝ freq^0.75 → 高频词多被当负例，降低其影响

## 一字一画面
- **负** —— 像**警察认嫌疑人**：真正的同伙（正例）站在一起，再拉几个路人（负例）混进去，训练警察一眼分辨。（本质：判别式学习）
- **采** —— 像**彩票摇号机**：按词频加权摇号，高频词更容易被抽中当"假搭配"。（本质：加权随机采样）
- **噪声** —— 像**掺沙子的金粉**：真金粉是正例，掺的沙子是负例，筛子要学会区分。（本质：对比学习雏形）

## 面试话术
- "负采样是为了解决Word2Vec里softmax分母要遍历整个词表的问题。核心思路是**正负对比**——把原来的多分类变成二分类：中心词和真实上下文词是正样本，再随机抽几个词作为负样本。训练目标就是让正样本的内积越来越大，负样本的内积越来越小。这样每轮只需要更新几个向量，而不是全词表，训练速度快了几个数量级。"

## 防混补丁
- 容易以为负采样是扔掉负样本。补丁：**负采样是主动构造负样本参与训练，不是丢弃；没有负样本就没有对比信号**。

## 追问预演
- Q：为什么采样分布要加0.75次方？
  - A：削弱高频词的主导地位，让中频词也有机会被采到。
- Q：负采样和层次softmax哪个更好？
  - A：负采样更灵活，实现简单，工业界更常用；层次softmax对词频差异大的场景有理论优势。
- Q：负采样个数K怎么选？
  - A：小数据集5-20个，大数据集2-5个即可；K越大负信号越强但越慢。
