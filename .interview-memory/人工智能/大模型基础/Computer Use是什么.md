# Computer Use 是什么

## 速记卡片
`操 | 操电脑，看屏点 | 计算机操作`

## 原始知识
- Computer Use：让AI能直接操作计算机的能力（Anthropic Claude首推）
- 核心能力：
  - **操**（操作）- 控制鼠标和键盘
  - **电脑**（计算机）- 操作真实电脑环境
  - **看**（看屏幕）- 读取屏幕内容（截图→视觉理解）
  - **屏**（屏幕）- 获取屏幕截图
  - **点**（点击）- 模拟鼠标点击、拖拽、输入
- 应用场景：自动化测试、数据处理、网页操作、软件使用
- **易混点**：Computer Use vs RPA
- **追问点**：怎么保证安全？怎么防止误操作？

## 顺口溜
**操电脑，看屏点**

## 单字锚点 + 防崩推导
- **锚点字**：**操**
- **扩展口诀**：**操电脑，看屏点**
- **为什么选它**：Computer Use的核心就是"操作计算机"，"操"是灵魂。
- **防崩推导**：
  - 操（操作）→ AI操作计算机
  - 电脑 → 操作真实电脑环境
  - 看 → 看屏幕截图
  - 屏（屏幕）→ 获取视觉信息
  - 点（点击）→ 模拟鼠标点击和键盘输入

## 一字一画面
- **操** —— 像**远程控制**：你在公司控制家里的电脑。（本质：AI操作计算机）
- **看** —— 像**看监控**：通过摄像头看屏幕内容。（本质：截图+视觉理解）
- **屏** —— 像**电视机**：AI看着屏幕决定按哪个按钮。（本质：屏幕截图输入）
- **点** —— 像**点外卖**：看着菜单，点击下单。（本质：模拟鼠标点击）

## 面试话术
- 问Computer Use → "Computer Use是让AI直接操作计算机的能力，口诀是**操电脑，看屏点**。**操**——控制鼠标和键盘；**电脑**——操作真实计算机环境；**看**——读取屏幕截图；**屏**——获取屏幕视觉信息；**点**——模拟点击、拖拽、输入。流程是：AI看屏幕截图→理解当前状态→决定下一步操作→执行鼠标/键盘动作→再看屏幕→循环。Anthropic的Claude是这方面做得最好的。"

## 防混补丁
- 容易和"RPA"搞混。RPA是录制回放固定流程，Computer Use是AI自主决策。补丁：**RPA是复读机，Computer Use是真人操作**。"
