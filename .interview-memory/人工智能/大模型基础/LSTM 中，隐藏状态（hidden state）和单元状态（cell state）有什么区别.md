## 速记卡片
`藏 | 藏深隐浅分两路 | C 深藏长期记忆，h 浅露短期输出`

## 原始知识
- **单元状态 C_t**：长期记忆通道，线性更新
- **隐藏状态 h_t**：短期输出表示，非线性变换
- **C_t 类似传送带**：信息直传，少变形
- **h_t 类似传声筒**：经过门控过滤后对外发声
- **C_t 携带远距信息**：梯度不易消失
- **h_t 携带即时语义**：每步都输出给下一层/时刻

## 顺口溜
深藏久，浅露新，C 管记忆 h 管声。

## 单字锚点 + 防崩推导
- **锚点字**：藏
- **扩展口诀**：藏深隐浅分两路，C 藏久忆 h 显今时
- **为什么选它**："藏"字精准对应单元状态的核心特质——把长期记忆"深藏"起来，不轻易外露；而隐藏状态是"显"出来的部分
- **防崩推导**：
  1. 想到"藏"→藏起来的 vs 露出来的
  2. 藏起来的是 C_t（cell state）：像海底电缆，信息在深处平稳传输，不受表面风浪影响
  3. 露出来的是 h_t（hidden state）：像海面浮标，每一时刻都可见，但只反映过滤后的信息
  4. 关系：h_t = o_t ⊙ tanh(C_t)，h_t 由 C_t 经输出门过滤得来
  5. 作用差异：C_t 解决长期依赖，h_t 承担即时表示和输出职责

## 一字一画面
**藏** —— 像**深海中的绿色保险箱被金色锁链拴住**：保险箱（C_t）在海底 3000 米沿隧道缓缓滑行，外表冰冷无波；每到一个站点，一位戴银色面具的守卫（输出门 o_t）打开小窗，只让箱内文件的一小部分摘要浮出水面，变成海面的橙色浮标（h_t）闪烁。（本质：C_t 深且稳，h_t 浅且活）

## 面试话术
"隐藏状态和单元状态最大的区别是角色分工不同。单元状态 C_t 是 LSTM 的‘长期记忆通道’，它通过线性更新把信息从序列开头传到结尾，负责解决长距离依赖问题。隐藏状态 h_t 则是‘短期输出表示’，它由单元状态经过输出门过滤后再做非线性变换得到，既传给下一个时间步，也作为当前层的输出。简单说，C_t 深藏记忆，h_t 显露行为。"

## 防混补丁
- **C_t 有 tanh 吗？**：C_t 更新本身无 tanh，只有候选态 C̃_t 有 tanh；h_t 计算时 tanh(C_t) 才出现
- **C_t 能直接输出吗？**：不能，必须经过 o_t 和 tanh 变成 h_t 才能对外
- **RNN 只有 h_t**：普通 RNN 没有 C_t，所以长距离记忆差
- **双向 LSTM**：有正向和反向两组 h_t，通常拼接
- **防崩**：C 是 cell（细胞/单元），细胞核深藏遗传信息→C_t 深藏长期记忆

## 追问预演
Q：为什么 C_t 用线性更新就能缓解梯度消失？
A：因为梯度在 C_t 的反向传播路径上是元素级乘法和加法，没有反复矩阵乘法导致的梯度指数级缩小。

Q：h_t 和 C_t 的维度一样吗？
A：通常一样，都是隐藏层维度，但它们语义完全不同。

Q：实际应用中我们用 h_t 还是 C_t 做下游任务？
A：一般用 h_t，因为它是门控过滤后的精炼表示；C_t 太"raw"，包含大量未经筛选的信息。
