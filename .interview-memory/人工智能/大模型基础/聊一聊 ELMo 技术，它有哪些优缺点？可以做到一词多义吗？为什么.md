# 聊一聊 ELMo 技术，它有哪些优缺点？可以做到一词多义吗？为什么？

## 速记卡片
`境 | 境双向，游语义 | 动态词向量`

## 原始知识
- **ELMo**：Embeddings from Language Models，基于双向LSTM
- **核心思想**：词向量不是固定的，根据上下文动态生成
- **结构**：两层双向LSTM，从左到右+从右到左
- **优点**：解决一词多义，上下文相关，语义丰富
- **缺点**：LSTM串行慢，特征抽取能力弱于Transformer，难以并行
- **一词多义**：能做到，因为同一词在不同上下文产生不同向量
- **为什么**：双向LSTM把整句信息编码进每个词的表示
- **易混点**：ELMo不是真正的"双向"同时编码，是两个单向拼接

## 顺口溜
**境双向，游语义**

## 单字锚点 + 防崩推导
- **锚点字**：**境**
- **扩展口诀**：**境动双，义不同**
- **为什么选它**："境"=语境，是ELMo的灵魂，一词多义靠的就是语境
- **防崩推导**：
  - 境 → 语境 → 词向量随上下文变化
  - 双 → 双向LSTM → 左看右看
  - 游 → 游标滑动 → 每个词都拿整句信息
  - 语义 → 同一词不同语境 → 不同向量 → 一词多义解决
  - 缺点 → LSTM慢、不能并行、特征弱

## 一字一画面
- **境** —— 像**变色龙随环境变色**：同一词在不同句子里"变色"。（本质：上下文相关）
- **双** —— 像**双眼看东西**：左眼从左看，右眼从右看，合起来立体。（本质：双向编码）
- **游** —— 像**游标卡尺滑动测量**：每滑到一个词，就把整句信息夹进去。（本质：动态编码）
- **义** —— 像**"bank"在水里是河岸，在金融里是银行**：同一单词两幅画像。（本质：一词多义）

## 面试话术
- "ELMo的核心是**境双向，游语义**。它用双向LSTM给每个词生成动态向量，同一个词在不同上下文里向量不一样，所以能做到一词多义。比如'bank'在河边和银行完全是两个向量。"
- "优点是解决了Word2Vec静态向量的多义问题；缺点是LSTM串行计算慢，而且特征抽取能力后来被Transformer超越了。"

## 防混补丁
- 容易和BERT的"真双向"搞混。ELMo是**两个单向LSTM拼接**，BERT是**多头自注意力真双向**。补丁：**ELMo是左眼+右眼分开看，BERT是全景相机同时拍**。

## 追问预演
- **问**：ELMo和Word2Vec最本质的区别是什么？
  - **答**：Word2Vec是查表（静态），ELMo是跑模型（动态），同一词输入不同句子输出不同向量。
- **问**：ELMo的词向量怎么用到下游任务？
  - **答**：把双向LSTM各层隐状态加权求和，拼接到下游模型输入上。
