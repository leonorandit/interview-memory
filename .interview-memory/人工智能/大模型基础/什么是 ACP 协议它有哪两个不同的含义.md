## 速记卡片
`协 | 一手协议一手通 | 双层含义莫混淆`

## 原始知识
- **含义一**：Agent Communication Protocol
- **指向**：Agent 间通信标准
- **含义二**：AI Computing Platform /  Anthropic 相关协议
- **指向**：算力调度或模型交互协议
- **共同点**：都是连接不同模块的"桥梁"
- **加粗易混点**：语境决定含义，**千万别张冠李戴**

## 顺口溜
ACP 两副脸，通信与算力。

## 单字锚点 + 防崩推导
- **锚点字**：协
- **扩展口诀**：协字两边站，Agent 与算力
- **为什么选它**："协"是协议的协，直接对应 Protocol，同时谐音"携"，有连接之意
- **防崩推导**：协 → 协议 → 两个领域都在用 → 大模型生态里先看上下文

## 一字一画面
**协** —— 像 **十字路口的交警**：左手指挥 Agent 车队互相通信，右手引导算力车辆有序通行（本质：ACP 是不同语境下的"交通指挥规则"）

## 面试话术
ACP 协议在大模型语境下有两个常见含义，一个是 Agent Communication Protocol，指 Agent 之间的通信标准；另一个可能指 AI 计算平台相关的协议。面试时遇到最好先确认上下文，别直接默认某一个。

## 防混补丁
- 遇到缩写先问上下文
- Agent 通信侧重消息格式，算力协议侧重资源调度
