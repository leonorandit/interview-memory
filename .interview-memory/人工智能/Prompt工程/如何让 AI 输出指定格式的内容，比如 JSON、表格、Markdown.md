# 如何让 AI 输出指定格式的内容，比如 JSON、表格、Markdown？

## 速记卡片
`格 | 格式框，示例绑 | 给框架+给例子`

## 原始知识
- **明确指定格式**：直接说"请输出JSON"
- **提供结构模板**：给出字段名和类型
- **给示例（Few-shot）**：写一段样板让AI照抄
- **用角色设定**："你是一位数据整理专家"
- **加约束条件**："不要解释，只输出代码"
- **分步骤拆解**：先列提纲，再填内容
- **易混点**：只说"用JSON"常出错，必须给字段定义

## 顺口溜
**格式框，示例绑**

## 单字锚点 + 防崩推导
- **锚点字**：**格**
- **扩展口诀**：**格框示绑**
- **为什么选它**："格"是格式的核心，听到"格"立刻想到"给框架"
- **防崩推导**：
  - 格 → 格式 → 先定框架（JSON/表格/Markdown）
  - 框 → 给结构模板（字段名、类型、层级）
  - 示 → 给示例（Few-shot，AI最会照抄）
  - 绑 → 绑约束（角色+条件+输出范围）

## 一字一画面
- **格** —— 像**填表格的格子**：先画好横竖线，再往里填字。（本质：结构化框架）
- **框** —— 像**乐高说明书**：每一步给图给编号，按图拼就行。（本质：模板引导）
- **示** —— 像**字帖描红**：给一行红字，照着描出黑字。（本质：示例学习）
- **绑** —— 像**给风筝系绳**：飞再高也限定范围。（本质：约束输出）

## 面试话术
- "让AI输出指定格式，核心是**格式框，示例绑**。先明确说输出JSON还是表格，再给字段结构模板，最后补一两个示例让AI照抄，基本不会错。"
- "只说一句'输出JSON'很容易字段名乱起，所以我习惯把字段名、类型甚至示例值都写好，AI的准确率能高很多。"

## 防混补丁
- 容易和"结构化Prompt"搞混。格式控制是**输出层**，结构化Prompt是**输入层**。补丁：**输入结构化为了清晰，输出格式化为了可用**。

## 追问预演
- **问**：如果AI还是不按格式输出怎么办？
  - **答**：加约束语"严格遵守上述格式，不要添加解释"，或者用系统提示词锁死角色。
- **问**：Markdown表格和JSON哪个更容易让模型理解？
  - **答**：JSON对机器最友好，Markdown表格对人最友好，看下游用途。
