## 速记卡片
`匠 | 提示如工匠，雕琢出华章 | 人设计输入，激发模型潜能`

## 原始知识
- Prompt Engineering 是设计输入的艺术
- 目标：让大模型输出更准、更稳、更可控
- 核心方法：角色设定、少样本示例、CoT、分隔符
- **不修改模型参数**，只优化输入
- 核心价值：低成本、高效率、快速迭代
- **易混**：不是微调，是零成本优化

## 一字一画面
**匠** —— 像**木匠握刻刀**：左手按住红木块（Prompt），右手刻刀（Engineering）细细雕琢，木屑纷飞中露出金色花纹（优质输出），工作台上有三把工具：角色牌、示例卡、分隔尺（本质：通过精心设计输入来释放模型能力）

## 面试话术
"Prompt Engineering 就是像工匠一样精心设计和打磨给模型的输入指令，通过角色设定、示例引导、思维链这些方法，让模型在不重新训练的情况下输出更好的结果。核心价值是低成本、见效快。"

## 防混补丁
| 概念 | 区别 |
|------|------|
| PE vs 微调 | PE 不改参数，微调改权重 |
| PE vs 模板 | PE 是动态设计，模板是固定格式 |
| 好 Prompt 标准 | 清晰、具体、有边界、有示例 |

## 追问预演
- Q: Prompt Engineering 和 Fine-tuning 怎么选？
  A: 样本少、任务变、要快试 → PE；样本多、要稳定、长期用 → 微调。
- Q: 怎么评估 Prompt 好坏？
  A: 看输出一致性、准确率、是否遵循格式约束。
