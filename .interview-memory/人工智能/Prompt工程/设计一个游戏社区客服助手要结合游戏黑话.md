# 设计一个游戏社区客服助手（要结合游戏黑话）

## 速记卡片
`黑 | 黑知角，情工圈 | 游戏客服六字诀`

## 原始知识
- **黑（黑话理解）**：内置游戏黑话词典（gg/afk/buff/debuff/肝/氪/欧非/开荒/farm/pvp/pve），让玩家觉得"这客服懂行"
- **知（知识库/RAG）**：接入游戏 FAQ、攻略、版本公告、活动规则，确保回答基于事实
- **角（角色扮演）**：用游戏 NPC 口吻或社区老玩家语气，增强沉浸感（如"勇士，你的装备该强化了"）
- **情（情绪安抚）**：识别玩家情绪等级，对愤怒/破防玩家先降温再解决问题
- **工（工具调用）**：查战绩、查活动进度、一键提交工单、举报、发放补偿码
- **圈（社区氛围）**：识别引战/晒欧/恶意言论，引导正向讨论，维护社区秩序
- **易混点**：游戏客服 ≠ 普通 FAQ，核心差异是角色沉浸和黑话共情
- **追问点**：黑话词典怎么构建、情绪识别怎么做、如何防止 NPC 角色越界

## 顺口溜
**黑知角，情工圈**

## 单字锚点 + 防崩推导
- **锚点字**：**黑**
- **扩展口诀**：**黑知角，情工圈**
- **为什么选它**："黑"是游戏客服最独特的标签——懂黑话是进入玩家语境的第一张门票
- **防崩推导**：
  - 黑 → 游戏黑话 → gg/afk/buff/肝氪欧非 → 让玩家觉得"自己人"
  - 黑后想底气 → 知 → RAG 知识库 → FAQ/攻略/公告 → 答得准
  - 知后想方式 → 角 → 角色扮演 → NPC 口吻/老玩家语气 → 有沉浸感
  - 玩家不爽时 → 情 → 情绪安抚 → 先降温再解决 → 防投诉
  - 解决问题需 → 工 → 工具调用 → 查战绩/工单/补偿 → 能闭环
  - 最后想环境 → 圈 → 社区氛围 → 识别引战/引导正向 → 保生态

## 一字一画面
- **黑** —— 像**游戏公会里的老玩家**：一开口就是"这波 buff 叠满了，别 afk，赶紧肝副本"。（本质：黑话是信任桥梁）
- **知** —— 像**游戏 Wiki 站**：打开就能查 BOSS 机制、装备掉落、活动规则。（本质：RAG 知识库兜底）
- **角** —— 像**酒馆里的 NPC 接待员**："冒险者，欢迎来到新手村，有什么可以帮你的？"（本质：角色扮演增强沉浸）
- **情** —— 像**公会会长调解矛盾**："兄弟先别气，这波确实运营锅，我帮你看看补偿进度。"（本质：情绪优先于事实）
- **工** —— 像**游戏内的 GM 面板**：查战绩、发补偿、封禁账号、一键提交工单。（本质：工具调用闭环）
- **圈** —— 像**论坛版主巡逻**：看到引战帖置顶劝架，晒欧过度的提醒节制。（本质：社区氛围维护）

## 面试话术
- "设计游戏社区客服助手，我总结为六字诀**黑知角，情工圈**。黑是内置游戏黑话词典，让玩家觉得'这客服懂行'；知是 RAG 接入游戏知识库，覆盖 FAQ、攻略、公告；角是角色扮演，用游戏 NPC 或老玩家口吻，增强沉浸感；情是情绪安抚，尤其对破防玩家先降温；工是工具调用，查战绩、查活动、一键提交工单；圈是社区氛围维护，识别引战言论并引导正向讨论。六招组合才能做出'不像机器人'的游戏客服。"

## 防混补丁
- 容易把游戏客服做成普通 FAQ 机器人。游戏客服的核心差异是**懂黑话、有角色、能共情**。补丁：**普通客服答问题，游戏客服进角色**。

## 追问预演
- **Q: 游戏黑话词典怎么构建？**
  - A: 从游戏论坛/社区爬取高频术语，运营团队人工审核，定期更新版本新词（如新赛季黑话），用 few-shot 让模型理解语境。
- **Q: 情绪识别怎么做？**
  - A: 基于关键词（骂/退/坑/骗）+ 标点（多感叹号/问号）+ 模型情感分类，分三级：正常/抱怨/愤怒，对应不同安抚策略。
- **Q: NPC 角色会不会越界？**
  - A: 设置护栏（Guardrails），禁止承诺未上线内容、禁止泄露未公开数据、涉及充值退款必须转人工。
