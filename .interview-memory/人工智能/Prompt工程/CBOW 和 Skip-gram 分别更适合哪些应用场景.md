# CBOW 和 Skip-gram 分别更适合哪些应用场景？

## 速记卡片
`预 | 预上下文，跳罕见 | 上下文vs目标词`

## 原始知识
- **CBOW**：用上下文预测中心词（Continuous Bag of Words）
- **Skip-gram**：用中心词预测上下文（Skip-gram）
- **CBOW优势**：训练快，对高频词效果好，适合小数据集
- **Skip-gram优势**：对低频词、罕见词效果好，适合大数据集
- **CBOW适用**：语义相对平滑的任务，如文本分类
- **Skip-gram适用**：需要细粒度词关系的任务，如推荐、类比推理
- **易混点**：谁预测谁容易记反

## 顺口溜
**预上下文，跳罕见**

## 单字锚点 + 防崩推导
- **锚点字**：**预**
- **扩展口诀**：**预众快，跳稀细**
- **为什么选它**："预"=预测，点明两者都是预测任务，同时谐音"域"有范围感
- **防崩推导**：
  - 预 → 预测 → CBOW是**上下文预测中心词**
  - 上下文 → 多词打包输入 → 训练快 → 适合高频常见词
  - 跳 → Skip-gram → **中心词跳出去预测周围**
  - 罕见 → 对低频词也认真学 → embedding更精细
  - 记忆锚：CBOW像"大家猜一个"，Skip像"一个人猜大家"

## 一字一画面
- **预** —— 像**猜谜游戏**：给线索猜谜底。（本质：预测任务）
- **上下文** —— 像**全班一起举手投票**：人多力量大，常见答案准。（本质：CBOW聚合上下文）
- **跳** —— 像**一个人点名周围朋友**：中心人物喊一圈名字。（本质：Skip-gram发散预测）
- **罕见** —— 像**博物馆里认真看冷门展品**：不因为人少就敷衍。（本质：低频词精细学习）

## 面试话术
- "CBOW和Skip-gram的核心区别是预测方向相反。CBOW是**上下文预测中心词**，像大家投票猜一个，训练快，对高频词好；Skip-gram是**中心词预测上下文**，像一个人点名周围，对罕见词学得细，适合大数据集。"
- "实际选型上，数据量小、追求速度选CBOW；数据量大、需要精细词关系选Skip-gram。"

## 防混补丁
- 容易记反"谁预测谁"。补丁：**CBOW是Bag， bag里装很多上下文词，一起猜中间那一个；Skip是跳，从中间跳到四周**。

## 追问预演
- **问**：负采样对哪个更有用？
  - **答**：对Skip-gram更有用，因为Skip-gram输出层维度大（词汇表大小），负采样能大幅加速。
- **问**：Word2Vec的窗口大小怎么选？
  - **答**：小窗口（2-5）学语法关系，大窗口（5-10）学语义关系，任务偏语义就调大。
