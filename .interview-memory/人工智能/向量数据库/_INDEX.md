# 向量数据库 领域速记索引

| 锚点字 | 知识点 | 口诀 | 类型 |
|--------|--------|------|------|
| 库 | 什么是向量数据库 | 向库存，语义搜 | 因果型 |
| 流 | 向量数据库的工作流程 | 存建查，返结果 | 流程型 |
| 算 | 向量数据库中的HNSW、LSH、PQ | 图哈希，量化压 | 列举型 |
| 索 | 向量数据库原理是什么 | 向存索，近邻找 | 因果型 |
| 距 | 向量数据库中的向量搜索方法 | 角直格，三度量 | 对比型 |
| 近 | 向量数据库中的ANN是什么 | 近近似，精换速 | 因果型 |
| 选 | 向量数据库选型 | 开云本，规性成 | 因果型 |
| 选 | 用的什么向量数据库Qdrant和Milvus有什么区别 | 选Q轻，选M重 | 对比型 |
| 量 | 讲讲你用的向量数据库？数据量级是多大？性能如何？遇到过性能瓶颈吗 | 量选索延写混 | 列举型/因果型 |

## 知识链路图

```
基础层：向库存，语义搜（库）
           ↓
原理层：向存索，近邻找（索）
           ↓
索引层：图哈希，量化压（算）
           ↓
搜索层：角直格，三度量（距）+ 近近似，精换速（近）
           ↓
流程层：存建查，返结果（流）
           ↓
选型层：开云本，规性成（选）+ 选Q轻，选M重（选）
           ↓
实践层：量选索延写混（量）
```
