# 全参数微调 vs LoRA vs QLoRA

## 速记卡片
`秩 | 全开贵，秩冻省，量四低 | 参不参`

## 原始知识
- **全参数微调**：更新模型**全部参数**，资源消耗最大，效果上限最高，易**灾难性遗忘**
- **LoRA**：**冻结**原权重，只训练旁路的**低秩**矩阵 (A/B)，参数量降千倍，效果接近全参数
- **QLoRA**：**4bit量化**模型权重 + LoRA，分页优化器+双量化，单卡可训大模型，推理需反量化
- **易混点**：LoRA不是压缩模型，是冻结原模型只加小模块；QLoRA不是训练量化后的模型，是量化存储+全精度LoRA训练
- **追问点**：为什么低秩有效？r怎么选？QLoRA分页优化器？
